NBA

ClutchCast AI

NBA win-probability platform that engineers 85 game-state features from play-by-play data and benchmarks four models head-to-head (a rule-based baseline, logistic regression, random forest, and a PyTorch neural net) on held-out games split by game ID to prevent leakage, then auto-selects the top performer as the live champion model for real-time inference.

Stack
Python, pandas, scikit-learn, PyTorch, Streamlit, Plotly, Flask, SocketIO, nba_api
Evaluation
Ranked by Brier score, log-loss, ROC-AUC, and accuracy on 28,463 held-out plays across 60 games from a 300-game season sample
Result
PyTorch neural net won on Brier score (0.180), the primary calibration metric, reaching 71.5% accuracy and a 0.81 ROC-AUC as the live champion model
